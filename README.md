# Excel Sales Data Analysis Dashboard

## 📊 Overview
This project demonstrates end-to-end data analysis using Microsoft Excel, including data cleaning, transformation, and dashboard creation.

## 🎯 Objective
To clean and analyze raw sales data and build an interactive dashboard that provides insights into sales performance, customer behavior, and product trends.

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- XLOOKUP
- Excel Formulas
- Charts & Dashboard Design

## 📁 Dataset
The dataset contains sales transactions including:
- Customer data
- Product categories
- Sales amounts
- Dates

## 🔧 Data Cleaning Steps
- Removed duplicates
- Handled missing values
- Standardized column formats
- Created calculated columns

## 📈 Analysis & Features
- Built pivot tables for aggregation
- Used XLOOKUP to merge and retrieve data
- Applied formulas for calculated metrics
- Designed an interactive dashboard with filters

## 📊 Dashboard Insights
- Top-performing products
- Sales trends over time
- Customer segmentation
- Regional performance
